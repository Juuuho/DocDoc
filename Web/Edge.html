Chromium

웹 브라우저 분석은, 사용자가 사용한 인터넷 사용 기록을 확인할 수 있습니다. 방문한 URL과 다운로드한 파일, Cookie 정보 등을 확인할 수 있습니다.

본 문서에서 서술하는 브라우저는 Microsoft Edge와 Chrome 입니다.

두 브라우저 모두 

 

Web History

사용자가 방문한 웹 사이트의 정보를 의미합니다.

 

 

Cache

웹 사이트에 접속할 시 방문 사이트에서 자동으로 받은 데이터를 의미합니다.

 

 

Download File List

사용자가 임의로 선택하여 저장한 파일들을 의미합니다.

 

 

Cookie

사용자의 정보를 저장하여 접속 상태를 유지하기 위해 사용되는 파일입니다.
