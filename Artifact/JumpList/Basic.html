JumpList

Jumplist란, Windows OS 7부터 새로 등장한 아티팩트입니다. 작업 표시줄 등에 고정된 아이콘을 우 클릭 했을 시, 최근 작업한 항목 혹은 사용자가 직접 고정한 항목 등을 확인할 수 있는데 이것이 바로 Jumplist입니다. 

 

Jumplist의 중요성

해당 아티팩트를 분석한다면 사용자의 컴퓨터 사용 기록을 추적 가능합니다. 더욱이, 사용자가 직접 삭제하지 않는 이상 OS 설치 시점부터 분석 시점까지 로그를 삭제하지 않고 지속적으로 저장하고 있기에 분석의 가치가 높습니다.

 

Jumplist의 경로

최근에 사용한 목록(Recent), 사용자가 고정시킨 목록(Pinned)

%UserProfile%\AppData\Roaming\Microsoft\Windows\Recent\AutomaticDestinations

자주 사용되는 목록(Frequent)이나 작업(Task) 목록

%UserProfile%\AppData\Roaming\Microsoft\Windows\Recent\CustomDestinations

  • 각 경로는 응용프로그램 별로 고유의 ID를 가져 정보를 관리합니다.
