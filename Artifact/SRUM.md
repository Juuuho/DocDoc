# SRUM
SRUM이란 System Resource Utilization Monitor의 약자로, Windows 8 부터 지원되는 기능입니다. 명칭 그대로 시스템의 자원 활용을 감시하는 아티팩트입니다. 응용 프로그램의 실행 여부, 경로, 배터리 사용량과 백그라운드, 포어그라운드에서 사용한 데이터 용량, 원격 네트워크를 활용해 전송한 데이터의 양 등 매우 다양한 정보들이 저장되어 있습니다. 응용 프로그램의 명칭과 경로, 해당 프로그램이 사용한 자원 등을 종합적으로 저장하고 있기 때문에 분석의 가치가 높습니다.


## 아티팩트 학습목표
분석자는 해당 아티팩트를 분석하여 사용자가 사용한 네트워크 자원 사용 정보, CPU 자원 사용 정보, 데이터 사용 정보 등을 알 수 있습니다.  이를 통해 사용자나 공격자가 사용하였던 응용 프로그램 실행 정보를 파악하고 추적할 수 있습니다.
